# 01 — Bookshelf API

**Concepts covered:** REST API design, path parameters, query parameters, request body (`Body()`), HTTP methods (GET, POST, PUT, DELETE), case-insensitive filtering, in-memory data operations.

**Project:** A fully functional CRUD API built with FastAPI to manage a collection of books — supporting retrieval by title, author, and category, alongside create, update, and delete operations.

**Status:** Complete
