# 02 — Movie Vault API

**Concepts covered:** REST API design, path parameters, query parameters, float-based filtering, request body (`Body()`), HTTP methods (GET, POST, PUT, DELETE), not-found handling, in-memory CRUD operations.

**Project:** A CRUD API built with FastAPI to manage a personal movie collection — supporting retrieval by title, genre, and director, a top-rated filter by minimum rating, and full create, update, and delete operations.

**Status:** Complete
