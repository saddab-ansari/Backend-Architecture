# 02 — Abstractions and Constructors

**Concepts covered:** Constructors (`__init__`), abstraction, `__str__` dunder method, multi-file architecture, object instantiation, iteration over object lists.

**Project:** Enemy Stats CLI — a command-line demo that instantiates multiple enemy objects and simulates a game-style stat display using core OOP principles.

**Key Note:** Understand Abstraction or Abstract data type as an ATM - you just press some buttons and swipe your card and money comes out, We don't really know what is happening in inside it.
